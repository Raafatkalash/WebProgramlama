# Fitness Center Yönetim ve Randevu Sistemi

Bu proje, Web Programlama dersi kapsamında geliştirilen bir **ASP.NET Core MVC** web uygulamasıdır.  
Amaç, bir spor salonu (fitness center) için üyelik, antrenör ve randevu yönetimini sağlamaktır.

## Proje Özeti

- Spor salonu tanımlamaları (çalışma saatleri, hizmet türleri, ücretler)
- Antrenör (eğitmen) yönetimi (uzmanlık alanları, çalışma saatleri)
- Üye kayıt ve giriş işlemleri
- Üyelerin uygun antrenör ve hizmet için randevu alabilmesi
- Randevu çatışmalarının (aynı saat, aynı antrenör vb.) kontrol edilmesi
- REST API ile raporlama (LINQ sorguları ile veri filtreleme)
- Yapay zekâ entegrasyonu ile kişiye uygun egzersiz/antreman önerileri

## Kullanılan Teknolojiler

- ASP.NET Core MVC
- C#
- Entity Framework Core, LINQ
- SQL Server / PostgreSQL
- HTML5, CSS3, Bootstrap 5
- JavaScript, jQuery
